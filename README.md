# led_device_driver
device driver for LED

# 概要
RaspberryPi3ModelB上でLED三個を光らせるプログラム

# 説明
このデバイスドライバはキーボードのrで赤色のLED、bで青色のLED、yで黄色のLEDを点灯させることが出来、dで全消灯します。
